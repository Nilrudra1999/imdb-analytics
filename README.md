# IMDb Analytics

<b>Author:</b> Nilrudra Mukhopadhyay<br>
<b>Project Type:</b> Data warehousing and EDA project - independently built<br>

## Project Overview

Business question:

- As a studio trying to release a movie in 2026 what would make the most amount of money/box office success?

Tech Stack:

- Python
- Microsoft SQL express server
- Microsoft SQL sever management studio (SSMS)
- Visual studio code
- PowerBI
- Git bash/Github

## Data Structures and Organization

Data Sources:

- [OMDb API open IMDb database, IMDb ID based](https://www.omdbapi.com/)
- [Free IMDb API, IMDb ID based](https://imdbapi.dev/)
- [Publicly available IMDb dataset, ID fragment](/data/imdb_movie_ids.txt)
